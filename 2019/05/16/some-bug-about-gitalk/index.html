##可是还是出现了“未找到相关的 Issues 进行评论，请联系xxx初始化”的情况

其实这里配置已经没有问题了，确实是要初始化，如果你点击登陆转跳回主页而且并没有登陆gitgub，那么极有可能是hexo皮肤的问题
